# Prompts structure

이 레포는 "코어(불변) + 프로젝트 오버레이(가변)" 형태로 프롬프트를 관리합니다.

## 코어(Core)

- 위치: .ai/prompts/core/
- 목적: 어떤 프로젝트에서도 동일하게 유지되는 개발팀 운영 규칙
- 원칙: 가능한 한 수정하지 않습니다(템플릿의 공통 자산)

## 오버레이(Overlay)

- 위치: .ai/project/overlay.md
- 목적: 프로젝트별로 달라지는 부분만 얇게 덧씌우는 파일
- 예: 스택/폴더 구조/도메인 규칙/금지사항/예시 입력과 출력

## 사용 방법(권장)

1. 코어 프롬프트는 그대로 둡니다.
2. 프로젝트를 만들 때마다 overlay.md만 수정합니다.
3. 에이전트를 실행할 때는 "코어 + overlay"를 함께 입력 컨텍스트로 제공합니다.
